React 中父组件向子组件传值

  • 在父组件中给子组件上添加 属性项 content={item} ，属性值 item，为要传递的数值。
  • 在子组件中通过 {this.props.content} 来接收父子间传过来的值。
  • 在父组件中给子组件上添加 属性项 deleteItem={this.deleteItem.bind(this)} ，属性值 this.deleteItem.bind(this)，为要传递的操作函数。
  • 在子组件中通过 {this.props.deleteItem} 来接收父子间传过来的操作函数。
  • 注意：子组件不能直接操作父组件的数据，可以通过传递父组件的操作函数，从而调用修改父组件中的数据。
