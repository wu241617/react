React 生命周期：在某一时刻，可以自动执行的函数

  • Initialization：初始化阶段 constructor 中，props 和 state 写入。
  • Mounting：虚拟 DOM 挂载阶段 componentWillMount ---> render (props 和 state 变化时触发) ---> componentDidMount
  • Updation：更新阶段
    1. props发生改变：componentWillReceiveProps(位于子组件，不在顶级组件中) ---> shouldComponentUpdate ---> componentWillUpdate ---> render ---> componentDidUpdate
    2. states发生改变：shouldComponentUpdate ---> componentWillUpdate ---> render ---> componentDidUpdate
  • Unmounting：卸载阶段 componentWillUnmount
  • 注意：componentWillReceiveProps(位于子组件，不在顶级组件中)，组件第一次存在于dom中，函数是不会被执行的，如果已经存在于dom中，函数才会被执行。
