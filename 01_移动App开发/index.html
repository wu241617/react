移动App开发

  • 什么是移动App开发，通俗的理解，就是把开发web网站的技术（html + css + js），通过某种方式，移植到移动App开发上使用，这种利用web开发技术进行移动端开发体验的方式，叫做混合移动App开发。
  • 原生开发：它的英文单词（NativeApp）,指的是使用IOS Android 官方提供的工具，开发平台，配套语言进行手机App开发的方式。
  • 混合开发（Hybird）：使用前端已有的技术，在搭配一些相关的打包编译技术，就能够开发出一个手机App,安装到手机中进行使用。
  • App，可安装的应用程序；它的分类：按照平台来分：PC端和移动端。按照功能来分：游戏和应用。
  • App概念：App是（Application的缩写），意思是，可安装的应用程序，优点是流畅稳定，基本上一些App都可以脱网运行，用户体验好。缺点是不能跨平台。
  • Web概念：特指那些基于浏览器的web网站。优点，可以跨平台（浏览器天生就是跨平台）。缺点：没有App流畅，不稳定，受限于网速和网络。
